# 共通アプリケーションアーキテクチャのテンプレート

このディレクトリは、`iac-generate` スキルが AWS/GCP/Azure のインフラストラクチャを生成する際にリファレンスとして使用できる、定義済みの Terraform アーキテクチャパターンのテンプレートを格納することを目的としています。

ここに配置される参照実装の例：
1. `three-tier-web.tf.template`
2. `serverless-api.tf.template`
3. `microservices.tf.template`

コードを生成する際、AI はここに含まれる標準的なベストプラクティスをエミュレートするように努める必要があります。
